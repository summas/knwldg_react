# 概要
リポジトリknowledgeAppをAPIサーバーとして作成した記事一覧を表示させるSPAアプリケーション。
このアプリケーションは作成都合上別のリポジトリで作成したものであり、knowledgeAppに組み込んである。

# 機能一覧
- 記事一覧表示機能（React)
- カテゴリーによる条件抽出（React)
- ページネーション(React, Material-ui)

#  使用技術
- React 16.13.1, Typescript 3.9.5
- Redux
- Material-UI
